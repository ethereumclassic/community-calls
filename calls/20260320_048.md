## Agenda

Before we get to Olympia, some thigns to sort out with 1121 

### EIP-7935: Set Default Gas Limit to 60M

EIP-7935 proposes increasing Ethereum's default gas limit from 36M to 60M via client configuration changes. This is currently included in the ECIP-1121 spec. Should it be deferred from ECIP-1121?

### ECIP-1121 Fork Naming

ECIP-1121 currently references the name "Olympia," which conflicts with the ECIPs 1111–1115 proposal that already uses that name. Should 1121 have its own distinct fork name? Seeking input from the community and ECIP authors.

### Olympia Discussion

The [Olympia upgrade discussion (#530)](https://github.com/orgs/ethereumclassic/discussions/530) has continued since the last call. A key development is the discovery of a contract deployment with admin key control over 1559 fund distribution — previously undisclosed information. The Olympia authors have not yet presented a solution to the bootstrap problem raised in earlier calls. How should the community evaluate this?

### New ECIPs: 1116, 1117, 1118, 1119

Several new ECIPs were recently merged. [ECIP-1116](https://github.com/ethereumclassic/ECIPs/pull/572) proposes a BASEFEE miner distribution with a 95/5 split. [ECIPs 1117–1119](https://github.com/ethereumclassic/ECIPs/pull/573) cover futarchy DAO governance, funding and streaming mechanisms, and treasury sanctions compliance. What are the community's initial thoughts on these proposals?

### ETC Nexus Updates
